# ? VitePress 博客管理器 GUI 使用指南

完整的图形用户界面版本，无需命令行，更加友好和直观。

## 启动 GUI

### Windows
```bash
# 方法1：双击运行脚本
双击 run_gui.bat

# 方法2：命令行启动
python blog_manager_gui.py
```

### Linux/macOS
```bash
# 方法1：运行脚本
chmod +x run_gui.sh
./run_gui.sh

# 方法2：直接使用Python
python3 blog_manager_gui.py
```

## 界面导览

### 1. ? 仪表板标签页

显示博客的实时统计信息：
- **? 文章总数** - 已发布的文章数
- **? 总字数** - 所有文章的总字数
- **? 平均字数** - 每篇文章的平均字数

功能：
- ? **刷新** - 手动更新统计信息

**文章列表区域：**
- 显示所有已创建的文章

**右键菜单（在文章上右键）：**
- ? **打开** - 用默认编辑器打开
- ?? **编辑** - 编辑文章
- ?? **删除** - 删除文章
- ? **复制路径** - 复制文章路径到剪贴板

**快速操作：**
- 双击文章直接打开编辑

### 3. ?? 管理标签页

**快速操作按钮：**
- ? **打开文章** - 打开选中的文章
- ?? **编辑文章** - 编辑选中文章

### 5. ? 发布标签页

**发布工作流：**

四个发布步骤，可单独执行或一键全部执行：

1. **? 构建站点**
   - 执行 `npm run docs:build`
   - 生成静态文件

2. **? 预览更改**
   - 检查要提交的更改
   - 可选步骤

3. **? 提交代码**
   - 提交所有更改到本地Git仓库
   - 使用自定义提交消息

4. **?? 推送远程**
   - 将代码推送到GitHub
   - 完成发布

**提交消息输入框：**
- 默认消息：`Update blog content`
- 可自定义提交消息

**一键发布：**
- ? **一键发布** - 自动执行所有步骤

**发布日志：**
- 显示每个步骤的详细信息
- 时间戳和状态指示

**项目信息：**
- 显示项目路径

**帮助信息：**
- 快速提示和常用快捷键
- 问题排查指南

## 常见工作流

### 工作流1：创建并发布新文章

```
1. 进入【?? 文章管理】标签页
2. 在【创建新文章】区域填写：
   - 文章标题：例如"Python最佳实践"
   - 分类：选择"tutorials"
   - 描述：添加简短描述
3. 点击【? 创建文章】按钮
4. 文章已创建，会自动打开编辑
5. 编写内容...
6. 进入【?? 预览】标签页，启动预览服务器
7. 访问 http://localhost:5173 查看效果
8. 进入【? 发布】标签页
9. 输入提交消息（可选）
10. 点击【? 一键发布】
11. 等待发布完成！
```

### 工作流2：编辑现有文章

```
1. 进入【?? 文章管理】标签页
2. 在文章列表中找到要编辑的文章
3. 双击或右键选择【?? 编辑】
4. 在默认编辑器中修改内容
5. 保存文件
6. 进入【?? 预览】标签页验证效果
7. 进入【? 发布】发布更新
```

### 工作流3：快速发布多个更新

```
1. 在文章上进行多个编辑
2. 进入【?? 管理】标签页
3. 点击【? 构建项目】
4. 进入【? 发布】标签页
5. 点击【? 一键发布】
```

## 故障排除

### 问题：打开失败 "UnicodeDecodeError"
**解决方案**：确保文档使用UTF-8编码

### 问题：预览服务器无法启动
**解决方案**：
1. 检查是否安装了Node.js和npm
2. 运行 `npm install` 安装依赖
3. 检查5173端口是否被占用

### 问题：发布失败
**解决方案**：
1. 检查Git是否已安装
2. 检查Git用户信息是否配置
3. 检查GitHub权限设置

### 问题：GUI启动失败
**解决方案**：
1. 确保Python版本 >= 3.7
2. 确保tkinter已安装（通常随Python一起安装）
3. 检查依赖的blog_manager.py是否在同目录

## 系统要求

- **Python** 3.7 或更高版本
- **tkinter** (通常随Python一起安装)
- **Node.js** 和 **npm** (用于VitePress预览和构建)
- **Git** (用于发布)
